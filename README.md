- `Vue 2.7.15`
- `@vue/cli 5.0.8`
- `webpack 5.89.0`

## git commit message 规范

| 类型     | 解释           | 示例（多个提交内容用;风格）      |
| -------- | -------------- | -------------------------------- |
| **feat** | 新功能、新特性 | feat: 新增/迁移登录功能          |
| **fix**  | 修改 bug       | fix: 修复登录密码错误问题        |
| **docs** | 文档修改       | docs: 修改 README.md, 服务器部署 |
